bool モード

bool モードでは、真偽値を生成します。コマンドの形式は次の通りです：

python -m randog bool [PROP_TRUE] [common-options]

引数とオプション

  • PROP_TRUE (optional, default=0.5):

    • True を生成する確率。

  • common-options

    • 共通オプション

実行例

最も簡素な例は次の通りです。True または False を 50% ずつの確率で出力します。

python -m randog bool

たいてい、1つ生成しただけでは足りないので、次のようにして複数回出力することになるでしょう。

# Repeat 10 times
python -m randog bool -r 10

# Generate list which contains 10 values
python -m randog bool -L 10

出力を他の言語のプログラムで処理する場合などでは小文字で出力する必要があるかもしれません。その場合、次のように json 形式で出力すると目的の形式で得られます。

# Repeat 10 times
python -m randog bool -r 10 --json

# Generate list which contains 10 values
python -m randog bool -L 10 --json
