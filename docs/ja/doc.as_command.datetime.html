datetime モード

datetime モードでは、日時を生成します。コマンドの形式は次の通りです：

python -m randog datetime [MINIMUM MAXIMUM] [--iso | --fmt FORMAT] [common-options]

引数とオプション

  • MINIMUM (optional):

    • 最小値。指定がない場合の動作は、randdatetime の仕様に準ずる。

  • MAXIMUM (optional):

    • 最大値。指定がない場合の動作は、randdatetime の仕様に準ずる。

  • --iso (optional):

    • 指定した場合、ISO-8601 形式で出力する。

  • --fmt FORMAT (optional):

    • 指定した場合、指定したフォーマットで出力する。フォーマットは strftime や strptime の形式 で指定する。

  • common-options

    • 共通オプション

実行例

最も簡素な例は次の通りです。日時を出力します。

python -m randog datetime

次の例のように、生成する値の範囲を指定することができます：

# generates a value between 2021-01-01T00:00:00 and 2021-01-01T12:00:00
python -m randog datetime 2021-01-01 2021-01-01T12:00:00

デフォルトではpythonの標準的な形式で出力されますが、次のようにオプションを指定することで出力形式を strftime や strptime と同じ形式 で指定したフォーマットやISOに変更できます：

# generates a value with ISO-8601 format
python -m randog datetime --iso

# generates a value with the specified format
python -m randog datetime --fmt '%Y/%m/%d %H:%M'

たいてい、1つ生成しただけでは足りないので、次のようにして複数回出力することになるでしょう。

# Repeat 10 times
python -m randog datetime -r 10

# Generate list which contains 10 values
python -m randog datetime -L 10 --json --fmt '%Y/%m/%d %H:%M'

注釈

datetime モードでは日付と時刻を同時に生成します。一方のみを望む場合、date モード や time モード を利用してください。

また、ある時点を指す時刻ではなく経過時間を生成したい場合は timedelta モード を利用してください。
