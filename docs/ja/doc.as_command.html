コマンドとして実行

randog はコマンドとして実行することもできます。コマンド実行の基本的な形式は次の通りです。

python -m randog <MODE> ...

生成したい値に応じたモードを <MODE> に指定し、より詳しい条件を指定するためにさらに引数を指定してください。

モード

下記のモードを使用することができます。各モードで使用できる引数やオプションはそれぞれのモードの詳細ページで確認できます。
