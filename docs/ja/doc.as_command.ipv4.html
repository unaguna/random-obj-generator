ipv4 Mode

ipv4 モードでは、IPv4 アドレスを生成します。コマンドの形式は次の通りです：

randog ipv4 [NETWORK ...] [--fmt FORMAT] [common-options]

警告

オプション --fmt は Python 3.9.0 以上でのみ使用できます。

引数とオプション

  • NETWORK (optional):

    • 生成するアドレスの空間。複数指定した場合、アドレス生成のたびにネットワークが抽選される。

  • --fmt FORMAT (optional):

    • IPv4Address.__format__ のドキュメント で記載された出力フォーマット

    警告

    オプション --fmt は Python 3.9.0 以上でのみ使用できます。

  • common-options

    • 共通オプション

実行例

最も簡素な例は次の通りです：

randog ipv4

次の例のように、ネットワークを指定することができます。

# generate an IPv4 address in 203.0.113.1-203.0.113.254
randog ipv4 203.0.113.0/24

# generate an IPv4 address in 198.51.100.1-198.51.100.254 or in 203.0.113.1-203.0.113.254
randog ipv4 198.51.100.0/24 203.0.113.0/24

出力フォーマット (二進法等)

以下のように IPv4Address.__format__ のドキュメント に記載されている出力フォーマットを使用することができます。

# output in binary numerical system such as '11000000000000000000001011000100'
randog ipv4 --fmt b

# output in hexadecimal such as 'C00002B7'
randog ipv4 --fmt X
