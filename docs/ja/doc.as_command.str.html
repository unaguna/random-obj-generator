str モード

str モードでは、文字列を生成します。コマンドの形式は次の通りです：

randog str [--length LENGTH] [--charset CHARSET] [--fmt FORMAT] [common-options]

randog str [--regex REGEX] [--fmt FORMAT] [common-options]

警告

この文字列生成では、暗号学的に安全であることは保証されません。認証用トークンの生成など、セキュリティ要件があるケースでは、secrets モジュール等の使用を検討してください。

引数とオプション

  • --length LENGTH (optional):

    • 生成される文字列の文字列長。--length 5 のように整数値を指定することや、--length 3:8 のように範囲を指定することができる。

  • --charset CHARSET (optional):

    • 生成される文字列が含む文字

  • --regex REGEX (optional):

    • 生成される文字列が満たす正規表現。--length や --charset と同時に使用することはできない。

  • --fmt FORMAT (optional):

    • 書式指定ミニ言語 で記載された出力フォーマット

  • common-options

    • 共通オプション

実行例

最も簡素な例は次の通りです：

randog str

次の例のように、文字列長や文字種を指定することができます。

# generate a string of length 16
randog str --length 16

# generate a string of length 8 - 16, which uses chars "0123456789abcdef"
randog str --length 8:16 --charset 0123456789abcdef

警告

この文字列生成では、暗号学的に安全であることは保証されません。認証用トークンの生成など、セキュリティ要件があるケースでは、secrets モジュール等の使用を検討してください。

また、生成される文字列を正規表現で指定することができます。

# generate a string of length 10 - 20, which uses numeric chars and alphabets
randog str --regex '[a-zA-Z0-9]{10,20}'

注釈

正規表現を使用した文字列生成には rstr パッケージを使用します。pip install rstr などであらかじめインストールする必要があります。

警告

正規表現を使用した文字列生成には rstr パッケージを使用します。使用する場合、rstr のライセンス条項を確認して従ってください。

出力フォーマット (右寄せ等)

以下のように出力フォーマットを 書式指定ミニ言語 で指定することができます。

# output string such as '      zy7q'
randog str --length 4 --fmt '>10'
