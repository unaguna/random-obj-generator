time モード

time モードでは、時刻を生成します。コマンドの形式は次の通りです：

python -m randog time [MINIMUM MAXIMUM] [--iso | --fmt FORMAT] [common-options]

引数とオプション

  • MINIMUM (optional):

    • 最小値。詳細は こちら。指定がない場合の動作は、randtime の仕様に準ずる。

  • MAXIMUM (optional):

    • 最大値。詳細は こちら。指定がない場合の動作は、randtime の仕様に準ずる。

  • --iso (optional):

    • 指定した場合、ISO-8601 形式 で出力する。

  • --fmt FORMAT (optional):

    • 指定した場合、指定したフォーマットで出力する。フォーマットは strftime や strptime の形式 で指定する。

  • common-options

    • 共通オプション

MINIMUM と MAXIMUM の表現方法

引数 MINIMUM, MAXIMUM はそれぞれ以下の記法で指定できます。

  • now: 現在時刻を意味します。

  • ISO-8601 形式。例: 11:22:33.000000, 11:22:33

  • timedelta の簡易書式 と時刻の和。例: now+2h, 01:00:00-1h30m

  • timedelta の簡易書式 (時刻の項を省略)。例: +2h:, -30m

    • MINIMUM と MAXIMUM のうちのもう一方が指定されていて時刻の項が省略されていないなら、これはそのもう一方に timedelta を加算した時刻を意味します。

    • MINIMUM と MAXIMUM のうちのもう一方が指定されていないか、そちらも時刻の項が省略されているなら、これは現在時刻に timedelta を加算した時刻を意味します。

注釈

-30m のような簡易表現を使用できますが、- で始まるため、オプション引数として解釈されエラーを引き起こしてしまいます。これを防ぐため、以下の例のように -- の後に指定する必要があります。

# valid (example for usage with --repeat)
python -m randog time --repeat 10 -- -30m +30m

# invalid (unknown option -30m)
python -m randog time --repeat 10 -30m +30m

実行例

最も簡素な例は次の通りです。日付を出力します。

python -m randog time

次の例のように、生成する値の範囲を指定することができます：

# generates a value between 00:00:00 and 12:00:00
python -m randog time 00:00:00 12:00:00

# generates a value between 1 hour ago and 1 hour later
python -m randog time -- -1h +1h

# generates a value between 12 hours ago and now
python -m randog time -- -12h

# generates a value between now and 30 minutes later
python -m randog time +30m

注釈

+1h 等の表現については MINIMUM と MAXIMUM の表現方法 を参照してください。

出力フォーマット (ISO-8601 等)

デフォルトではpythonの標準的な形式で出力されますが、次のようにオプションを指定することで出力形式を strftime や strptime と同じ形式 で指定したフォーマットや ISO-8601 形式 に変更できます：

# generates a value with ISO-8601 format
python -m randog time --iso

# generates a value with the specified format
python -m randog time --fmt '%H:%M'

繰り返し生成

たいてい、1つ生成しただけでは足りないので、次のようにして複数回出力することになるでしょう。

# Repeat 10 times
python -m randog time -r 10

# Generate list which contains 10 values
python -m randog time -L 10 --json --iso

注釈

time モードでは日付は生成されません。日付も込みで生成したい場合、datetime モード を使用してください。

また、ある時点を指す時刻ではなく経過時間を生成したい場合は timedelta モード を利用してください。
