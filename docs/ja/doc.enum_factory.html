Enum factory

列挙子を生成する factory を作成することができます。

他の factory と同様に、dict factory も2つの方法 (randenum, from_example) で作成することができます。

randenum を使用して作成

randenum を使用する場合、以下の例のようになります。

>>> import enum
>>> import randog.factory
>>> from randog.factory import DictItem

>>> class MyEnum(enum.Enum):
...     one = 1
...     two = 2

>>> # create a factory
>>> factory = randog.factory.randenum(MyEnum)

>>> generated = factory.next()
>>> assert generated in MyEnum

注釈

randchoice を使用しても同様の factory を作成できます: randog.factory.randchoice(*MyEnum)

デフォルトでは各列挙子が同じ確率で生成されますが、引数 weights を指定することで確率を偏らせることもできます。

>>> import enum
>>> import randog.factory
>>> from randog.factory import DictItem

>>> class MyEnum(enum.Enum):
...     one = 1
...     two = 2

>>> # a function which returns the probabilities of each value
>>> def weights(value):
...     if value == MyEnum.one:
...         return 0.8
...     elif value == MyEnum.two:
...         return 0.2

>>> # create a factory
>>> factory = randog.factory.randenum(MyEnum, weights=weights)

>>> generated = factory.next()
>>> assert generated in MyEnum

from_example を使用して作成

from_example を使用する場合、以下の例のようになります。

>>> import enum
>>> import randog.factory
>>> from randog import DictItemExample

>>> class MyEnum(enum.Enum):
...     one = 1
...     two = 2

>>> # create a factory
>>> factory = randog.factory.from_example(MyEnum.one)

>>> generated = factory.next()
>>> assert generated in MyEnum
