List factory

list 型のオブジェクトを生成する factory を作成することができます。この factory では list の各要素も毎回ランダムに生成します。

多くのケースでは、用途に応じた調整が必要となるため from_example よりも randlist を使用します。

>>> import randog.factory

>>> factory = randog.factory.randlist(
...     randog.factory.randint(0, 100),
...     randog.factory.randstr(),
... )

>>> generated = factory.next()
>>> assert isinstance(generated, list)
>>> assert isinstance(generated[0], int)
>>> assert isinstance(generated[1], str)

注釈

もし1つの factory から複数のレコードを作成したい場合、randlist よりも iter を使うべきかもしれません。

>>> import randog.factory

>>> factory = randog.factory.randdict(
...     id=randog.factory.randint(0, 999_999),
...     price=randog.factory.randdecimal(0, 1000, decimal_len=2),
... )

>>> # generate 200 records
>>> record_list = list(factory.iter(200))

コマンド実行 の場合も --list オプションや --repeat オプションを使用することで同様のことができます。

それぞれの要素の生成

それぞれの要素の factory を指定することができます。通常、i番目の要素の生成にはi番目に指定した factory が使用されますが、指定した factory の数よりも長い list を生成する場合、最後の factory が繰り返し使用されます。

>>> import randog.factory

>>> el_factories = [
...     randog.factory.randint(0, 100),
...     randog.factory.randstr(),
...     randog.factory.randbool(),
... ]
>>> factory = randog.factory.randlist(
...     *el_factories,
...     length=4,
... )

>>> generated = factory.next()
>>> assert isinstance(generated, list)
>>> assert len(generated) == 4
>>> assert isinstance(generated[0], int)    # by el_factories[0]
>>> assert isinstance(generated[1], str)    # by el_factories[1]
>>> assert isinstance(generated[2], bool)   # by el_factories[2]
>>> assert isinstance(generated[3], bool)   # by el_factories[2]

したがって、それぞれの要素が特別な意味を持たない典型的なリストを生成したい場合は、要素の factory を1つだけ指定してください。

>>> import randog.factory

>>> factory = randog.factory.randlist(
...     randog.factory.randint(0, 100),
...     length=4,
... )

>>> generated = factory.next()
>>> assert isinstance(generated, list)
>>> assert len(generated) == 4
>>> for el in generated:
...     assert isinstance(el, int)

長さ

生成されるリストの長さをランダムにすることもできます。

>>> import randog.factory

>>> factory = randog.factory.randlist(
...     randog.factory.randint(0, 100),
...     length=randog.factory.randint(3, 5),
... )

>>> # Repeat generation to make sure length is at random
>>> lengths = set(map(len, factory.iter(1000)))
>>> assert lengths == {3, 4, 5}

もし長さを指定しない場合、factory の数と同じ長さのリストが生成されます。

>>> import randog.factory

>>> el_factories = [
...     randog.factory.randint(0, 100),
...     randog.factory.randstr(),
...     randog.factory.randbool(),
... ]
>>> factory = randog.factory.randlist(
...     *el_factories,
... )

>>> generated = factory.next()
>>> assert isinstance(generated, list)
>>> assert len(generated) == len(el_factories)

tuple の生成

引数 type を使うことで、list ではなく tuple を生成することもできます。

>>> import randog.factory

>>> factory = randog.factory.randlist(
...     randog.factory.randint(0, 100),
...     randog.factory.randstr(),
...     type=tuple,
... )

>>> generated = factory.next()
>>> assert isinstance(generated, tuple)

注釈

randlist に渡す length を省略すると factory の個数と同じ長さとなるため、各要素が個別のスキーマを持つような典型的な tuple を生成する場合は length を指定する必要がありません。

警告

引数としてイテレーターを受け入れるような型であれば type に指定することができますが、set を指定すると要素の個数が保証されません。

from_example を使用して作成

from_example を使用する場合、以下の例のようになります。

>>> import randog.factory

>>> factory = randog.factory.from_example([1, "a"])

>>> generated = factory.next()
>>> assert isinstance(generated, list)
>>> assert isinstance(generated[0], int)
>>> assert isinstance(generated[1], str)

例として与えたリストの各要素は、生成するリストの各要素の例として使用されます。
