ログ出力

randog は、python が提供する標準的な方法 でログを記録します。そのため、記録されたログのストリーム出力やファイル出力は、あなたが標準的な方法で設定できます。ライブラリのドキュメント や HOWTO も参照してください。

注釈

randogをコマンドとして実行する際にログを出力する方法は、こちら をご覧ください。

注釈

randog のログ出力はあまり豊富でないため、あまり情報を得られないかもしれません。今後徐々に改善する予定です。

API

randog は下記のロガーを使用します。

  • randog: randog で使用する logger の root

    • randog.cmd: randog をコマンドとして実行する際に使用される logger

    • randog.factory: factory の生成や factory の機能で使用される logger

    • (適宜 logger を追加する)

注釈

randog のログを取得する場合、ロガー randog やルートロガー root にハンドラを設定することをお勧めします。今後ロガーを増やすかもしれないため、子ロガーごとに設定すると想定外の取捨選択をする可能性があるためです。
