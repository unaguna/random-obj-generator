クイックスタート

以下の手順で randog を試すことができます。

インストール

Python 3.8 以上を用意し、以下の pip コマンドで randog をインストールします。

pip install randog

最小構成での利用

生成したいオブジェクトの例を from_example 関数に渡すことで、オブジェクトをランダムに生成する factory を作ることができます。

>>> import uuid
>>> import randog.factory

>>> factory = randog.factory.from_example({"uuid": uuid.uuid4, "name": "", "age": 20})
>>> generated = factory.next()

>>> assert isinstance(generated, dict)
>>> assert isinstance(generated["uuid"], uuid.UUID)
>>> assert isinstance(generated["name"], str)
>>> assert isinstance(generated["age"], int)

この例では dict オブジェクトを生成していますが、リストや単なる文字列等を生成することもできます。
