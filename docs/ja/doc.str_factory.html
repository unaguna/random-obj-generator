Str factory

randstr を使用することで、文字列を生成する factory を作成できます。

注釈

from_example に文字列を例として渡しても str の factory を作成できますが、生成条件の指定ができません。

単純な factory

生成される値が文字列でありさえすればよい場合には、次のように引数を指定せずに randstr を使用します。

>>> import randog.factory

>>> # create a factory
>>> factory = randog.factory.randstr()

>>> generated = factory.next()
>>> assert isinstance(generated, str)

文字列長や文字種を指定する

引数 length や charset を使用することで、生成される文字列の長さや使用する文字種を指定することができます。

>>> import string
>>> import randog.factory

>>> # create a factory
>>> factory = randog.factory.randstr(length=8, charset=string.ascii_uppercase)

>>> generated = factory.next()
>>> assert isinstance(generated, str)
>>> assert len(generated) == 8
>>> assert set(generated) <= set(string.ascii_uppercase)

length に randint() を指定することで、文字列長をランダムにすることもできます。

>>> import string
>>> import randog.factory

>>> # create a factory
>>> factory = randog.factory.randstr(length=randog.factory.randint(10, 19))

>>> generated = factory.next()
>>> assert isinstance(generated, str)
>>> assert 10 <= len(generated) <= 19

charset には、使用する文字からなる文字列を指定します。通常は string モジュールの定数を指定しますが、自身で用意した文字列を指定することもできます。

>>> import string
>>> import randog.factory

>>> # create a factory
>>> factory = randog.factory.randstr(charset="AZ")

>>> generated = factory.next()
>>> assert isinstance(generated, str)
>>> assert set(generated) <= {"A", "Z"}

正規表現を指定する

正規表現によって、生成される文字列を指定することができます。

>>> import re
>>> import randog.factory

>>> # create a factory
>>> factory = randog.factory.randstr(regex=r"\d{3}-\d{4}-\d{4}")

>>> generated = factory.next()
>>> assert isinstance(generated, str)
>>> assert re.fullmatch(r"\d{3}-\d{4}-\d{4}", generated)

警告

引数 regex を使用する場合、length や charset は使用できません。

注釈

正規表現を使用した文字列生成には rstr パッケージを使用します。pip install rstr などであらかじめインストールする必要があります。

警告

正規表現を使用した文字列生成には rstr パッケージを使用します。使用する場合、rstr のライセンス条項を確認して従ってください。
