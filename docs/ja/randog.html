class randog.Example(*objs)

ベースクラス: Sequence

randog.generate_to_csv(factory: Factory, line_num: int, fp: TextIO, csv_columns: Sequence[str | Callable[[Mapping], Any]] | None, *, regenerate: float = 0.0, discard: float = 0.0, raise_on_factory_stopped: bool = False, linesep: str | None = None)

値をランダムに生成し、CSVとして出力する。

パラメータ:

  • factory (Factory) -- 値を生成する factory

  • line_num (int) -- 繰り返しの回数。ただし、引数 raise_on_factory_stopped が True でない場合、factory が停止すると size より少ないイテレーションで停止する。また、引数 discard が指定されている場合、イテレーションの回数が少なくなる場合がある。

  • fp (TextIO) -- CSVの出力先

  • csv_columns (list[str, Callable[[Mapping], Any]] | None) -- 各列の定義。文字列で定義した場合、生成されたオブジェクトからその文字列をキーとして値を取り出す。関数で定義した場合、生成されたオブジェクトを引数としてその関数を実行し、戻り値を使用する。

  • regenerate (float, default=0.0) -- factory が生成した値をそのまま使用せずに再生成する確率。このオプションは factory が完全にランダムでない値を生成する場合に効果がある。

  • discard (float, default=0.0) -- factory が生成した値をそのまま使用せずに廃棄する確率。廃棄が生じた場合、生成される数は size で指定した数より少なくなる

  • raise_on_factory_stopped (bool, default=False) -- True を指定すると、factory が StopIteration によって生成を停止した場合に例外 FactoryStopException を発する。False を指定するか何も指定しない場合、先述のケースで単にイテレーションを停止する。

  • linesep (str, optional) -- 指定した場合、CSVの行は指定した文字列で区切られる。
