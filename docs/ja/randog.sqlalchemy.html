randog.sqlalchemy package

randog.sqlalchemy.custom(example, **kwargs)

sqlalchemy 由来のオブジェクトの生成のための custom_func。

この関数は randog.factory.from_example のアドオンです。

from_example で sqlalchemy 由来の値を生成する factory を作成したい場合、この関数を引数 custom_func として指定します。

サンプル

>>> from sqlalchemy import Column, Integer, String
>>> from sqlalchemy.orm import declarative_base
>>> import randog.factory
>>> import randog.sqlalchemy
>>>
>>> Base = declarative_base()
>>>
>>> class User(Base):
...     __tablename__ = "user"
...
...     id = Column(Integer, primary_key=True, autoincrement=True)
...     name = Column(String)
>>>
>>> # specify `randog.sqlalchemy.custom` as `custom_func`
>>> factory = randog.factory.from_example(User,
...                                       custom_func=randog.sqlalchemy.custom)
>>> generated = factory.next()
randog.sqlalchemy.factory(model: Type[M], override_columns: Mapping[str, Factory | Any] | None = None, *, as_dict: bool = False, rnd: Random | None = None) → Factory[M] | Factory[dict]

ランダムにモデルインスタンスを生成する factory を返す。

パラメータ:

  • model (Type[Model]) -- sqlalchemy のモデルクラス

  • override_columns (Mapping[str, Union[Factory, Column, Any]], optional) -- この dict で値を指定したカラムについては、その値を使用して factory を生成し、model 内の column 定義は無視する。factory オブジェクトがこの dict の値に指定された場合、それがそのままフィールドの生成に使用される。それ以外が指定された場合、それを引数として from_example を実行することで得られる factory がフィールドの生成に使用される。

  • as_dict (bool, default=False) -- True の場合、モデルインスタンスの代わりに dict 型の値を生成する。

  • rnd (Random, optional) -- 乱数生成に使用するインスタンス

randog.sqlalchemy.factory_from_column(column: Column, *, rnd: Random | None = None) → Factory

カラムに合致する値をランダムに生成する factory を返す。

パラメータ:

  • column (Column) -- sqlalchemy のカラム

  • rnd (Random, optional) -- 乱数生成に使用するインスタンス
